:focus-within Función: Aplica estilos cuando un elemento, o cualquiera de sus hijos, está enfocado (recibe el foco del teclado o del ratón). Esto es útil para formularios o contenedores con múltiples elementos interactivos. Ejemplo: Un formulario que cambia el color del borde del contenedor cuando cualquiera de sus elementos está enfocado. Pseudoclase :focus-within
:empty Función: Selecciona elementos que no tienen ningún contenido (es decir, que están vacíos). Esto es útil para agregar estilos especiales a elementos que aún no han sido llenados o que no contienen hijos. Ejemplo: Un div que cambia de color si está vacío. html Copiar código Pseudoclase :empty
