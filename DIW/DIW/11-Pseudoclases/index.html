PSEUDOCLASES:

Sirven para dar estilos a los distintos estados que pueda tener un elemento

Se representa con selector: nombre_pseudoclase Ejemplo div: hoover

Link a w3 schools

Usos:

  • Para cambiar los estados de los enlaces; (orden):link, visited, hover,active...
  • Para cambiar estados de los items de las listas: first-child, :last child, only child, nn-child
  • Para cambiar estado de los formularios
  • Para cambiar estado de las filas y columnas de las tablas
