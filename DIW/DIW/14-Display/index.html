DISPLAY

No cambia el tipo de elemento, cambia como se visualizan los elementos en mi sitio web

Tipos de elementos html:

  • Bloque: block, elemento que empieza en una nueva linea, ocupa todo el ancho disponible
    1. Encabezados h1-h6
    2. parrafo
    3. formulario
    4. listas ul,ol, dl y sus li
    5. div
    6. table,td,tr,th
  • En linea: inline, ocupan solamente el tama√±o de su content, no les podemos dar ni ancho ni alto y van uno a continuacion de otro
    1. a enlaces
    2. imagenes img
    3. span

VALORES DISPLAY

  • none: no se muestra
  • visibility hace lo mismo que none pero deja el hueco del elemento
  • block: se muestra como si fuera un elemento en bloque
  • inline: se muestra como si fuera un elemento inline
  • inline-block: se muestra como si fuera medio bloque medio inline
